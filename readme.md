

# Concept 

```
wrap \
    {input_path} \
    --output_path {output_path} \
    --processor_path {processor_path} \
    --clean_output_path True
```
to generate wraps from input files using processors.

## Terminology
- input folder
- input files
- wraps folder
- wraps
- processors
